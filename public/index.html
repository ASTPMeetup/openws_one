The public JSON storage.

Free, open-source service to store JSON data through a RESTFul API.

No signup or authentication needed.

* It's easy to use!
Get started with one command

# The command below saves `{ "name": "Testing" }` in the `books` collections.

Protoype web & mobile apps faster

Don't spend time developing a back-end service to handle your data when prototyping your app anymore. OpenWs is free, open-source and it couldn't be easier to use.

OpenWs is a RESTFul service

OpenWs APIs support the following HTTP methods: POST, PUT, GET, DELETE.

Another important feature is that CORS is enabled, which means AJAX requests made from browser will be accepted by OpenWs. No more Cross-Domain AJAX request issues.

OpenWs is an open-source project

That means you can modify or extend OpenWs to your needs.

Check out the project page in Github.

Run it in your own machine if you prefer

OpenWs is built in Ruby on Rails and MongoDB. All the information on how to setup OpenWs locally can be found here.
